# Backend & AI Portfolio

## 1. 概要 (Overview)
本リポジトリは、バックエンド開発およびAIを活用したシステム開発に関する
学習成果と実装例をまとめたポートフォリオです。

REST API設計、データ処理、機械学習モデル統合などの経験を通じて、
信頼性・拡張性・保守性を意識したシステム開発の理解を示しています。

---

## 2. 目的 (Purpose)
私はバックエンドを中心に、クラウド／システム設計の能力を高め、
実務に使えるソフトウェアを構築できるエンジニアになることを目指しています。

このポートフォリオは、以下のようなスキル・経験を示すために構成されています：
- REST API の設計と実装
- Python を用いたバックエンド開発
- 機械学習モデル統合と実運用の視点
- フロントエンドからバックエンドへの API 利用設計
- シンプルで保守しやすいコード構造

---

## 3. プロジェクト一覧 (Projects)

### 3.1 AI Heart Disease Risk Prediction API
- 患者の健康データを使い、心臓病リスクを予測するバックエンド API システム
- Flask を用いた REST API と機械学習モデル統合
- API の入力検証、エラーハンドリング、安定性を重視した設計
- 詳細: [`ai-heart-disease/README.md`](./ai-heart-disease/README.md)

---

### 3.2 Habit Tracker Web Application
- 日常の習慣管理を目的としたフロントエンド中心の Web アプリ
- サーバーレス（localStorage）で構成され、レスポンシブ UI を実装
- シンプルで保守性の高いフロントエンド設計
- 詳細: [`habit-tracker-web/README.md`](./habit-tracker-web/README.md)

# Backend & AI Portfolio

## 1. Tổng quan (Overview)
Repository này là portfolio tổng hợp các kết quả học tập và ví dụ triển khai
liên quan đến phát triển backend và xây dựng hệ thống có ứng dụng AI.

Thông qua các trải nghiệm về thiết kế REST API, xử lý dữ liệu và tích hợp mô hình
học máy, portfolio thể hiện sự hiểu biết về phát triển hệ thống theo hướng
đảm bảo độ tin cậy, khả năng mở rộng và tính dễ bảo trì.

---

## 2. Mục tiêu (Purpose)
Tôi hướng tới trở thành một kỹ sư tập trung vào backend, có khả năng thiết kế
hệ thống và làm việc với môi trường cloud, đồng thời xây dựng được các phần mềm
có thể sử dụng trong thực tế.

Portfolio này được xây dựng nhằm thể hiện các kỹ năng và kinh nghiệm sau:
- Thiết kế và triển khai REST API
- Phát triển backend bằng Python
- Tích hợp mô hình học máy và tư duy vận hành thực tế
- Thiết kế cách frontend sử dụng API từ backend
- Xây dựng cấu trúc code đơn giản, dễ bảo trì

- Thiết kế chú trọng vào kiểm tra dữ liệu đầu vào, xử lý lỗi và tính ổn định
